# NodeMinecraftAutoAFK

**基于NodeJS的MinecraftAFK机器人**

## 一、使用

 1. 安装NodeJS环境
 2. 修改config.js的账号密码服务器地址（包括验证方式）
 3. 在Minecraft中保护自己并手持食物下线
 4. 运行main.js

## 二、依赖

推荐最新版本的node，低版本node不支持
